# 税务统筹计算器

![Tax Optimization Example](example.jpg)

通过穷举个人年度税前工资，找出在给定年度总盘子下税负最低的最优分配方案，并生成交互式可视化图表。

## 快速开始

### 安装依赖

本项目使用 [uv](https://docs.astral.sh/uv/) 作为 Python 包管理工具。

```bash
# 安装 uv（如果尚未安装）
curl -LsSf https://astral.sh/uv/install.sh | sh

# 克隆项目后，安装依赖
uv sync
```

### 运行脚本

```bash
# 使用默认参数运行
uv run python tax_optimizer.py

# 自定义参数运行
uv run python tax_optimizer.py -p 1500000 -d 24000 -s 5000
```

运行后会：
1. 在终端打印最优方案的详细分解
2. 生成 `tax_optimization.html` 交互式图表

## 命令行参数

| 参数 | 短参数 | 默认值 | 说明 |
|------|--------|--------|------|
| `--pool` | `-p` | 1000000 | 企业分配给个人的年度总盘子（元） |
| `--deduct` | `-d` | 18000 | 个人年度专项附加扣除总额（元） |
| `--step` | `-s` | 10000 | 穷举工资时的步长（元） |
| `--output` | `-o` | tax_optimization.html | 输出 HTML 文件名 |

### 示例

```bash
# 年度总盘子 200 万，专项扣除 3.6 万，步长 2000
uv run python tax_optimizer.py --pool 2000000 --deduct 36000 --step 2000

# 输出到指定文件
uv run python tax_optimizer.py -p 1500000 -o my_analysis.html
```

## 计算逻辑

### 1. 五险一金（个人缴纳部分）
- 缴纳比例：20%
- 缴费基数上限：400,000 元/年
- 公式：`ins = min(salary, 400000) × 0.20`

### 2. 个人所得税（IIT）
- 基础减除费用：60,000 元/年
- 应纳税所得额：`T1 = max(0, salary - 60000 - ins - deduct)`
- 按七级超额累进税率计算

| 应纳税所得额 | 税率 | 速算扣除数 |
|--------------|------|------------|
| ≤ 36,000 | 3% | 0 |
| ≤ 144,000 | 10% | 2,520 |
| ≤ 300,000 | 20% | 16,920 |
| ≤ 420,000 | 25% | 31,920 |
| ≤ 660,000 | 30% | 52,920 |
| ≤ 960,000 | 35% | 85,920 |
| > 960,000 | 45% | 181,920 |

### 3. 企业所得税（CIT）
- 小微企业实际税率：5%
- 公式：`CIT = (pool - salary) × 0.05`

### 4. 分红个税（DivTax）
- 税后利润分红税率：20%
- 公式：`DivTax = (pool - salary - CIT) × 0.20`

### 5. 总税负
`Total Tax = IIT + CIT + DivTax`

### HTML 图表

生成的交互式图表包含：
- **X 轴**：个人年度税前工资
- **Y 轴**：总缴税金额
- **悬浮提示**：完整税费分解、净到手金额
- **最优点标记**：红色星形标记显示最优方案

## 注意事项

1. 本脚本仅供学习参考，实际税务筹划请咨询专业税务顾问
2. 税率政策可能变化，请以最新法规为准
3. 五险一金比例和基数上限因地区而异，本脚本使用简化假设
